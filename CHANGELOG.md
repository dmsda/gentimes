# Changelog

All notable changes to the **GenTimes** project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-28

### Initial Release
- **Frontend**: Launched Next.js 16 App Router application.
- **CMS**: Deployed Strapi v5 with custom schema.
- **Features**:
  - Full article rendering with Markdown support.
  - Custom privacy-friendly Analytics system.
  - "Trending News" algorithm and UI.
  - Newsletter subscription with double-opt-in logic.
  - Comment system with moderation queue.
  - PWA support (Manifest + Service Worker).
  - Dark/Light mode toggle.
  - AI-ready SEO structure and JSON-LD.
- **Performance**:
  - Achieved LCP < 1.2s.
  - Implemented ISR caching strategy.
  - Optimized images with Sharp/WebP.

## [0.9.0] - 2025-12-20

### Beta
- Core news listing and article pages.
- Basic Strapi integration.
- Tailwind CSS design system implementation.
